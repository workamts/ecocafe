## Description

EcoCafé is a website that facilitates the purchase of organic coffee, education on sustainable practices, and the connection between consumers and responsible producers. The goal is to create a space where quality, ethics, and respect for the environment are the fundamental pillars.

## Features

- Catalog of certified organic coffees
- Detailed information about producers and processes
- Educational blog about sustainability and coffee
- Shopping cart and order management
- Intuitive and responsive interface

## Technologies Used

- **Frontend:** Html, Css, JavaScript
- **Version Control:** Git & GitHub
- **Additional Tools:** ESLint, Prettier, Jest

## Descripción

EcoCafé es una sitio web que facilita la compra de café ecológico, la educación sobre prácticas sostenibles y la conexión entre consumidores y productores responsables. El objetivo es crear un espacio donde la calidad, la ética y el respeto por el medio ambiente sean los pilares fundamentales.

## Características

- Catálogo de cafés ecológicos certificados
- Información detallada sobre productores y procesos
- Blog educativo sobre sostenibilidad y café
- Carrito de compras y gestión de pedidos
- Interfaz intuitiva y responsiva

## Tecnologías Utilizadas
